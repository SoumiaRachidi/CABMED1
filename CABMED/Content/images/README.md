## ?? Color Scheme

All custom SVG illustrations use the MEDX color palette:
- **Primary Blue**: `#2D3E6E`
- **Accent Orange**: `#FF6B4A`
- **Light Orange**: `#FFB4A0`
- **Soft Blue**: `#E8F0FF`
- **Light backgrounds**: `#D8E5FF`

## ?? How to Add New Illustrations

1. Place SVG files in `CABMED/Content/images/`
2. Use the `~/Content/images/filename.svg` path in views
3. Match the color scheme above for consistency
4. Keep file sizes small (optimize SVGs)
